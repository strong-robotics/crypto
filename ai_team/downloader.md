# Агент Downloader

## Цель
Собрать и обновить локальный кэш HTML‑документации внешних API в `./ai_team/docs/` без интерпретации содержания.

## Источники
- ./ai_team/_api_sources.md — список URL для загрузки (Jupiter/Helius/Solana и др.).
- mcp:fetch — для HTTP GET по URL и сохранения ответа как HTML.
- mcp:filesystem — для записи файлов в `./ai_team/docs/`.

## Правила
1. Прочитать все URL из `./ai_team/_api_sources.md` (по одному на строку или в списках).
2. Для каждого URL выполнить HTTP GET (через mcp:fetch) и сохранить ответ в `./ai_team/docs/`.
3. Имя файла: нормализованный URL: заменить `://` и `/` на `_`, добавить суффикс `.html`.
   - Пример: `https___dev.jup.ag_docs_tokens_v2.html`.
4. По умолчанию перезаписывать существующие файлы (режим refresh). Поддержать флаги:
   - only-missing — скачивать только отсутствующие.
   - list — вывести список целевых файлов, не загружая.
5. Вести простой манифест `./ai_team/docs/manifest.json` с полями: url, filename, size, fetched_at (ISO), status.
6. Не парсить содержимое — Downloader не делает выводов.

## Выходные артефакты
- Файлы `./ai_team/docs/*.html` — свежий кэш документации.
- `./ai_team/docs/manifest.json` — журнал загрузок.

## Типовой запуск
- «Загрузить все по списку»: выполнить через Codex CLI инструкцию из `_agents.md` для Downloader.
- «Обновить только отсутствующие»: запустить с флагом only-missing.
